My Projects & My Experiences

Autonomous Email Filtering System

I wrote a Python program that automatically deletes unwanted emails from specific organizations or with specific keywords.I made this system fully autonomous using GitHub Actions and YAML configuration. I used the IMAP protocol. Now the system runs automatically every morning at 10:00 AM Turkish time, scanning the specified organizations and keywords and destroying the emails.

Autonomous Underwater Vehicle Development

I have contributed to the development of the AUV of DDD organization. My work focused on motion control, mission automation, and computer vision. I develop software in Python and C++ using Pixhawk, PyMAVLink, and ROS. I implement PID-based control algorithms and use OpenCV for underwater object detection and visual processing. I also worked on integrating embedded systems and sensors to enable fully autonomous missions.
